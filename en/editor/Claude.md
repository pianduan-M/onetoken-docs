**Keep the application logged out (or sign out of your account first).**

1. Click **Menu** → **Help** → **Troubleshooting** → **Enable Developer Mode**
   <img src="/images/claude/1.png" />
   <img src="/images/claude/2.png" />

2. After completing the first step, a **Developer** menu will appear in the menu bar. Click **Configure Third-Party Inference**.
   <img src="/images/claude/3.png" />

3. By default, select **Gateway**. Enter the following details:
   - **Gateway base URL:** `https://onetoken.one/`
   - **Gateway API key:** Use the API Key obtained from the [Onetoken Platform](https://console.aihubmix.com/token).

   Leave other settings as they are, click **Apply locally**, and then click **Relaunch now** in the pop-up window.
   <img src="/images/claude/4.png" />
   <img src="/images/claude/5.png" />

4. Click **Continue with Gateway**.
   <img src="/images/claude/6.png" />

5. Select the model you wish to use from the bottom of the chat box to begin.
   <img src="/images/claude/7.png" />
