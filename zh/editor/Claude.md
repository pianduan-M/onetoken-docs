保持未登录状态（或先退出账号）

1. 点击菜单 → Help → Troubleshooting → Enable Developer Mode
   <img src="/images/claude/1.png" />
   <img src="/images/claude/2.png" />

2. 经过第一步的操作后菜单栏会多出一个 Developer 菜单， 点击 Configure Third-Party Inference
   <img src="/images/claude/3.png" />

3. 默认选择 Gateway 填写 Gateway base URL: https://onetoken.one/ 和 Gateway API key [Onetoken 平台](https://console.aihubmix.com/token) 获取的 API Key 其他不用动 点击 apply locally , 弹窗点击 relaunch now
   <img src="/images/claude/4.png" />
   <img src="/images/claude/5.png" />

4. 点击 Continue with Gateway
   <img src="/images/claude/6.png" />

5. 对话框下方选择要使用的模型，即可开始使用
   <img src="/images/claude/7.png" />
