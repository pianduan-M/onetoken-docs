# Kilo Code

> Kilo Code 是一款基于 LLM 的 AI 编程插件，能够帮助开发者更高效地编写、调试和优化代码。

## 插件配置

1. 运行 VS Code，安装 Kilo Code。

</Frame>

2. 点击 `Kilo Code` 插件图标，进入配置页面。

    <img src="https://mintcdn.com/aihubmix/hsnZJ3bKDY9b0fRA/images/%E5%BE%AE%E4%BF%A1%E5%9B%BE%E7%89%87_2026-03-25_154246_867.png?fit=max&auto=format&n=hsnZJ3bKDY9b0fRA&q=85&s=c6abe11536fdac010cc249e909d79542" alt="微信图片 2026 03 25 154246 867" width="2146" height="1384" data-path="images/微信图片_2026-03-25_154246_867.png" />
