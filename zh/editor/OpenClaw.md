> OpenClaw 是一个开源的智能体平台，可在你的本地机器上运行，并能通过你已经在使用的聊天应用来工作。

## 安装

<Note>
  安装前请确保 Node>=22
</Note>

执行安装指令

```shellscript theme={null}
npm install -g openclaw@latest
```

### 终端配置

打开终端，运行以下指令开始配置

```shellscript theme={null}
openclaw onboard
```
